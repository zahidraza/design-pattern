# **Design Patterns**

## **Gang of Four Design Patterns**

1. Creational Design Pattern
    - Abstract Factory
    - Builder
    - Factory Method
    - Prototype
    - Singleton
2. Structural Design Pattern
    - Adapter
    - Bridge
    - Composite
    - Decorator
    - Facade
    - Flyweight
    - Proxy
3. Behavioural Design Pattern
    - Chain of Responsibility
    - Command
    - Interpreter
    - Iterator
    - Mediator
    - Memento
    - Observer
    - State
    - Strategy
    - Template Method
    - Visitor
